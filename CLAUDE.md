# FIBA Labs Website

Company marketing website deployed at https://fibalabs.com.

## Tech Stack

- React 19 + Vite 7
- Tailwind CSS 3.4
- GSAP (ScrollTrigger animations)
- Lucide React icons
- Deployed on Cloudflare Pages (auto-deploy from GitHub on push to main)

## Related

- Brand assets: ~/OS/fiba labs assets/
- Course origin: ~/OS/courses/nick-saraev/gemini website builder/

## Development

- `npm run dev` -- local dev server
- `npm run build` -- production build to dist/
- Push to main branch triggers automatic Cloudflare Pages deployment

## Domain

- Production: https://fibalabs.com
- Preview: https://fiba-labs-website.pages.dev
- DNS managed via Cloudflare (same account as other fibalabs.com subdomains)
